# Proyecto Colaborativo DAW  
### Desarrollo Web – Control de Versiones con Git y GitHub

---

## Descripción del Proyecto
Este proyecto forma parte de la práctica colaborativa de Git y GitHub.  
El objetivo es simular el desarrollo de una web sencilla utilizando el trabajo en equipo, ramas, commits individuales y fusionar luego en en GitHub.

## Historial del Proyecto
Cada miembro del equipo ha trabajado en su propia rama, y ha hecho commits individuales y contribuyendo al desarrollo del proyecto.  
El administrador ha gestionado la integración final en la rama `main`.
